Boston 311 Data

311 is the non-emergency phone number and web portal for the City of Boston. Residents can use 311 to request City services, such as trash and recycling pickup, pothole repair, and more. The City of Boston makes this data available to the public, and we can use it to visualize trends in requests over time.

D3 is used to visualize the number of 311 requests over time. The 311 Service Requests dataset is used, which contains data on all 311 requests from 2015 to the present. The dataset can be downloaded as a CSV file. Alternatively the API can be used to retrieve the data programmatically.

Data Source: City of Boston

Going the Extra Mile

The above visualization shows the total number of the number of 311 requests over time. The total count is a combination of requests form the Citizens_Connect_App, City_Worker_App, Constituent_Call, Employee_Generated, twitter, and Self_Service. Consequently, Plotly is used to create the plot below which shows the breakdown of the total in to the different call sources. The visualization is interactive, and the user can zoom in to figure out the breakdown for a specific neighborhood. The visualization is also responsive, and will adjust to the size of the browser window.
